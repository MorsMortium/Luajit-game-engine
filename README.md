# Luajit-Game-Engine
3D game engine written in LuaJIT
## Prerequisites
LuaJIT, OpenGL, GSL, SDL
## Features
Rendering via SDL or OpenGL  
Multiple cameras  
Broad phase collision detection with sweep and prune  
Narrow phase collision detection with GJK and EPA  
More to come...
